# 🎬 Interactive World — Motion Graphic Story

An interactive video experience hosted on the web, featuring a short story told through motion graphics crafted in Adobe After Effects. Viewers can choose between two worlds, each with its own visual narrative.


## ✨ Features

- Interactive branching experience - slide to choose your world
- Short-form storytelling through motion graphics
- Smooth video playback embedded directly in the browser
- Fully self-contained, no external dependencies

## 🎨 Production

Videos were created using **Adobe After Effects**, featuring custom motion graphics, animation, and visual storytelling techniques.

## 🚀 Getting Started
Live at: https://tinhngo22.github.io/twoworlds/

No build tools needed. Simply open `index.html` in your browser:

```bash
# Clone the repo
git clone https://github.com/your-username/interactive-world.git

# Open in browser
open index.html
```

> ⚠️ For best results, use a modern browser (Chrome, Firefox, Edge) that supports HTML5 video playback.
